# ⚡ NAROK - UNPUMPER TOOL
**The ultimate binary file stripping tool for malware analysis and data cleanup.**

## ✨ Features
- 🚀 **Extreme Speed:** Uses `mmap` for kernel-level file access.
- 🌍 **Smart UI:** Automatically detects Windows language (TH/EN).
- 📦 **Universal:** Supports EXE, MP4, JPG, PNG, and more.
- 📏 **Dynamic Units:** Supports from Bytes up to Yottabytes (YB).

## 🛠️ Installation & Usage
1. Clone the repo: `git clone https://github.com/nelakunch-sketch/NAROK-UNPUMPER.git`
2. Run: `python unpumper.py`
3. Or just **Drag & Drop** your file onto the `.exe` version!
